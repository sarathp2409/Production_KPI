================================================================================
PAGINATION DATA COLLECTION - VALID VALUES REFERENCE
================================================================================

BOOK NAMES (book_name column):
----------------------------------------
  - TOI B-1
  - TOI B-2
  - TIMS B-1
  - TIMS B-2
  - TOI Supp-1
  - TOI Supp-2
  - TOI Supp-3

GNP PAGE TYPES (gnp_pages_type columns):
----------------------------------------
  - 2 Pg Jkt
  - 4 Pg Jkt - 2 diff clients
  - 4 Pg Vantage
  - 4 Pg Power Jkt
  - 8 Pg Super-pano
  - 6 Pg GNP
  - 8 Pg GNP

INNOVATION TYPES (innovation_1/2/3 columns):
----------------------------------------
  - Center Panorama
  - Bookmark
  - Seamless Panorama
  - Front Vantage
  - Reverse Flap
  - Fragrance
  - French-Window
  - Power Jacket
  - Others

BOOLEAN VALUES (balloon_printed, has_masthead, gnp_* columns):
----------------------------------------
  - Yes or TRUE or 1 = True
  - No or FALSE or 0 or empty = False

DATE FORMAT (issue_date column):
----------------------------------------
  - YYYY-MM-DD (e.g., 2023-01-15)

PLANT GNP PUBLICATIONS:
----------------------------------------
  Airoli: Mirror, MT, TIMS
  Baroda: (No GNP publications)
  Bhosari: ET, MT, TIMS
  Bommasandra: ET, TIMS, Mirror
  Butibori: (No GNP publications)
  Chemmencherry: ET, TIMS
  Chinhat: ET, NBT, TIMS
  Kandivali: ET, MT, NBT, Mirror, TIMS
  Manesar: ET, NBT, TIMS
  Nacharam: ET, TIMS
  Sahibabad: ET, NBT, TIMS
  Saltlake: ET, TIMS
  Trivandrum: (No GNP publications)
  Vejalpur: ET, TIMS

================================================================================
NOTES:
================================================================================
1. balloon_printed and has_masthead are only applicable for TOI B-2 and TIMS B-2
2. gnp_et_b1, gnp_et_b2, gnp_et_had_2_books are only for plants with ET publication
3. Report-level fields (remarks, GNP flags) only need to be filled in the first
   book row for each date - they will be ignored in subsequent rows
4. Competitor data must be imported AFTER in-house data for the same dates
